# Walkthrough - iOS App 摄像头翻转升级 (v2.5.0)

本版本为 CastNow Pro 增加了灵活的摄像头控制功能，支持在直播/广播过程中自由切换前后摄像头。

## 修改内容

### 1. 摄像头轨道定向管理
- **优化**：在 `_startBroadcast` 逻辑中，现在会独立识别并缓存摄像头视频轨道 (`_cameraTrack`)。
- **并发兼容**：这一改进确保了“翻转”操作只会影响摄像头画面，而不会干扰同时进行的屏幕共享流。

### 2. 硬件级无缝切换
- **核心逻辑**：新增 `_switchCamera` 方法，调用 `Helper.switchCamera` 实现硬件级的镜头翻转。
- **体感**：切换过程流畅，远程接收端无需重新连接即可实时看到镜头变换。

### 3. 增强型控制界面
- **UI 呈现**：在广播控制面板中，若检测到开启了摄像头，将自动显示一个“翻转”图标按钮 (`Icons.flip_camera_ios_rounded`)。
- **状态感知**：按钮仅在摄像头源被激活时才会出现，保持界面简洁。

## 验证结论
- [x] **功能验证**：已验证在“屏幕+摄像头”并发模式下，翻转镜头仅改变摄像头视角，屏幕共享保持稳定。
- [x] **异常处理**：增加了对切换失败的错误捕获逻辑。
- [x] **归档完成**：本实施方案已同步归档至 `dev_process/walkthrough_v2.5.0.md`。
